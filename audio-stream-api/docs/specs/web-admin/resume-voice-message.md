Feature: El administrador puede obtener un resumen del contenido del mensaje de voz, utilizando IA.

Goal:
Construir un endpoint para resumir el contenido del mensaje de voz, utilizando inteligencia artificial para evitar que en medio de un stream tenga que escuchar un audio.

Constraints:
- Además del resumen de audio, necesito poder identificar palabras clave tales como insultos provocaciones o cualquier cosa que Youtube pueda considerar como una causal de strikes.
